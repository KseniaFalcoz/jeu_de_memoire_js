Jeu de mémoire

Le jeu se compose de paires de cartes avec les mêmes images. Toutes les cartes sont mélangées, puis posées face cachée. Vous cliquez deux cartes. Si les images sont identiques, les cartes restent ouvertes. Si les images sont différentes, les cartes sont à nouveau retournées face cachée.

Vous devez ouvrir toutes les cartes.
Le gagnant est celui qui ouvre toutes les cartes le plus rapidement.
